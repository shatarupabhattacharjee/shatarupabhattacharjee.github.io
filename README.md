# Shatarupa's Artree Website

A vibrant, modern website for Shatarupa's art store and online painting classes.

## Features

- **Beautiful Design**: Vibrant color scheme inspired by your logo (coral red, teal, and golden yellow)
- **Responsive Layout**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive Gallery**: Filterable gallery to showcase your paintings by category
- **WhatsApp Integration**: Direct contact button for easy communication
- **Contact Form**: Integrated form that opens WhatsApp with pre-filled message
- **Modern UI/UX**: Smooth animations, hover effects, and professional styling

## Sections

1. **Header**: Logo, navigation, and tagline
2. **Hero**: Welcome section with call-to-action buttons
3. **About**: Information about your services and features
4. **Gallery**: Showcases your paintings with filtering options
5. **Services**: Details about custom paintings and online classes
6. **Contact**: Contact form and WhatsApp integration
7. **Footer**: Links and social media

## Setup Instructions

1. Replace the placeholder images in the gallery with your actual paintings
2. Update the WhatsApp number in the JavaScript file (search for "1234567890")
3. Add your social media links in the footer
4. Customize the contact information

## Customization

### Colors
The website uses your logo's color palette:
- Primary Coral: #E85A4F
- Primary Teal: #4A9B8E  
- Primary Gold: #F4A261
- Cream Background: #F7F3E9
- Dark Brown: #2D1B14

### WhatsApp Integration
To update your WhatsApp number:
1. Open `script.js`
2. Find all instances of "1234567890"
3. Replace with your actual WhatsApp number (with country code, no + sign)

### Gallery Images
Replace the placeholder images in the HTML file with your actual artwork URLs or local image files.
