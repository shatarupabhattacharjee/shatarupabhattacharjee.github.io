## Quick Solutions:

### Option 1: Remove Duplicates (Fastest)
Remove duplicate files with similar names:
- Keep one version of each artwork
- Remove "_paintings_" prefixed duplicates

### Option 2: Compress Images (Recommended)
Use online tools to compress:
- https://tinyjpg.com
- https://compressjpeg.com
- Target: Under 500KB per image

### Option 3: Resize Images
Reduce dimensions to 1920x1080 max:
- Most web images don't need full resolution
- Use GIMP, Photoshop, or online resizers

## Recommended Action:
1. Remove duplicate images first
2. Compress images > 1MB
3. Keep only best quality versions
